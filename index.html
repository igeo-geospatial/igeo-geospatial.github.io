Project Management

Create and organize research projects through intuitive interface

Georeferencing

⚠️ Data Storage Notice

All collected data is stored exclusively on your device using local storage (SQLite). The application does not transmit or backup data to external servers. Users are entirely responsible for data management and preservation. Uninstalling the application will permanently erase all collected data.

We recommend regular data backups through export features.
